Common ICT Support Issues & Fixes

Dashboard My Profile

Internet Connectivity

  • Check if the network cable is properly connected.
  • Ensure Wi-Fi is turned on and connected.
  • Restart the router or modem.
  • Use the Windows Network Troubleshooter.

Printer Issues

  • Check printer power and USB/network cable connection.
  • Ensure the correct printer is selected as default.
  • Restart the print spooler service from Services app.
  • Update or reinstall printer drivers.

Email Problems

  • Check internet connection.
  • Clear browser cache or try another browser.
  • Verify email credentials are correct.
  • Contact support if mailbox is full or locked.

Slow Computer Performance

  • Restart your computer to clear temporary files.
  • Uninstall unnecessary programs.
  • Run Disk Cleanup and check for malware.
  • Upgrade RAM or storage if needed.
